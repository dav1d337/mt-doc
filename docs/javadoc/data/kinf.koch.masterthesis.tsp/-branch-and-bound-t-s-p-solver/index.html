BranchAndBoundTSPSolver

(JVM) @MapScope class BranchAndBoundTSPSolver : TSPSolver (source)

Solves the Traveling Salesman Problem with a banch-and-bound-approach. By branching the problem in different subproblems and bounding the solutions, it will find the hamilton cycle inside an adjacency matrix in O(n^3).
